Detailed Description

UI class to handle password related activities.

UI class to displaying dialog boxes needed to perform activities like adding a password entry, changing a password entry, changing the master password etc.

void initializeList ( List  list)

Create the list of entries in the UI.

Creates the list of entries in the UI.

Each element shows the website name, but the password is not shown. The user can get the password by selecting the website whose password he wants and using the Copy button in the Edit menu. The password will be copied to the clipboard and will be ready to be pasted.

listThe list that will be initialized with the password entries.
